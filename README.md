# Pixel Assistant

Pixel Assistant is a retro-styled web application that combines several useful and fun tools in one place, all with a charming pixel-art aesthetic.

## Features

- **Mega Randomizer**: Generate random numbers, floats, or pick random items from lists
- **Pro Stopwatch**: Full-featured stopwatch with lap functionality
- **Smart Timer**: Countdown timer with notifications
- **Password Generator**: Create secure passwords with customizable options
- **Color Generator**: Generate random colors, gradients, and palettes
- **Number Guessing Game**: Classic "guess the number" game

## How to Use

Simply open the `index.html` file in any modern web browser. No installation or dependencies required!

### Randomizer
1. Choose between basic and advanced modes
2. Set your range or input custom values
3. Click "Generate" to get a random result

### Stopwatch
1. Click "Start" to begin timing
2. Use "Lap" to record split times
3. "Pause" stops the timer, "Reset" clears it

### Timer
1. Set hours, minutes, and seconds
2. Click "Start" to begin countdown
3. Receive notification when timer completes

## Customization

Change the color theme by clicking the theme buttons in the Stopwatch panel:
- Purple (default)
- Pink
- Green

## Browser Support

Pixel Assistant works in all modern browsers including:
- Chrome
- Firefox
- Safari
- Edge
